## Requirements to run
* Java 11 or latest
* Maven 3+

## General info
This project is a simple console application that provides ability to manages a list of products.

	
## Technologies
Project is created with:
* Java 11
* Spring Core 5
* Maven 3.5
* Git 2+
* JUnit 5
	
## Setup
To run this project, install and run it locally using maven:

```
$ mvn clean install
$ mvn exec:java
```
