# ListOfGoods

## Main topic mof project
It is just simple project which provides ability to manage the list of goods
	
## Technologies
Project is created with:
* Java 11
* Spring core 5.1.7
* Maven 3.5+
* Git 2+
* JUnit 5
	
## Deploy
To run this project, install and run it locally using maven:

```
$ mvn clean install
```
## Execute

```
$ mvn exec:java
```
